## ğŸ“ Customization

### Update Resume
Replace `assets/resume/resume.pdf` with your latest resume.

### Update Images
- Add project images to `assets/images/`
- Update image paths in `index.html`
- Add `og-image.jpg` (1200x630px) for social media previews

### Update Contact Form
The contact form uses EmailJS. Update the service ID and template ID in `js/port.js`:
```javascript
emailjs.send("YOUR_SERVICE_ID", "YOUR_TEMPLATE_ID", {...})
```
